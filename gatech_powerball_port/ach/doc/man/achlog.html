NAME

achpipe.bin - log ach messages  

SYNOPSIS

achlog [OPTIONS] channels...  

DESCRIPTION

Log ach channels to files Options:
-?,
Show help
-z,
Filter output through gzip
 

EXAMPLES

achlog foo bar
Log channels foo and bar
