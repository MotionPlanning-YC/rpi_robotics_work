NAME

achcat - ach debugging tool  

SYNOPSIS

achcat [OPTION...] [-p|-s] channel  

DESCRIPTION

Copies ach frames as string lines to/from stdio. Mostly intended as an ach testing tool.
-p CHANNEL-NAME,
Publish to a channel
-s CHANNEL-NAME,
Subscribe to a channel
-?,
Give this help list
