NAME

achcop - Watchdog for ach daemons  

SYNOPSIS

achcop [OPTIONS...] -- child-name [CHILD-OPTIONS]  

DESCRIPTION

Watchdog to run and restart ach child processes  

OPTIONS

-P pid-file,
File for pid of cop process (only valid with -r)
-p pid-file,
File for pid of child process
-o out-file,
Redirect stdout to this file (- keeps unchanged, no option closes)
-e err-file,
Redirect stderr to this file (- keeps unchanged, no option closes)
-d,
Detach and run in background
-r,
Restart failed children
-i ident
Log identifier (default: achop)
-f facility
Log facility {user|daemon} (default: user)
-v,
Make output more verbose
-?,
Give program help list
-V,
Print program version
 

EXAMPLES

achcop -rd -P /var/run/myd.ppid -p /var/run/myd.pid -- my-daemon -xyz
